# Docker Ruby Cookbook

Builds a docker image containing the following:

* ruby 2.1.2
* chruby

# Requirements

* [chef-docker cookbook](https://github.com/bflad/chef-docker)

# Usage

Include `recipe[docker-ruby::default]` in your run_list.
